# NEP-Silicides

Neuroevolution Potential (NEP) models for titanium-silicide (Ti-Si) systems, trained using [GPUMD/NEP4](https://github.com/brucefan1983/GPUMD).

This repository contains the training pipeline and validation benchmarks for machine-learned interatomic potentials targeting Ti-Si systems using the NEP4 formalism. Potentials are trained with a genetic-algorithm-based evolution strategy on first-principles data.
